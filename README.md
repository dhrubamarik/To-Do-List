📝 To-Do List Web App

A simple and elegant To-Do List application built using HTML, CSS, and JavaScript.
This project helps users manage daily tasks with an intuitive and visually pleasing interface.

🚀 Features
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
🎯 Interactive UI with clean design
💾 Persistent task state (if implemented with local storage)
📱 Responsive layout
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & layout
JavaScript – Functionality & DOM manipulation
